MyShop:

Desarrollado por mí, Paolo Saavedra.

Este proyecto de MyShop, consistió en crear una tienda web desde 0, sin el uso de librerias y/o Frameworks. Esto es para demostrar lo que sé sobre sus lenguajes vanilla.

Para este se uso las siguientes tecnologías:

- HTML5 : Estructura de las páginas.

- CSS3 : Diseño y decoración de las páginas.

- JavaScript : Funcionalidades Dinámicas, procesamiento de formularios, solicitudes a la API de la tienda web, manipulación de archivos JSON.

- PHP : Intermediario entre JavaScript y la base de datos(API), inclusión de scripts.

- APIs : API CRUD hecha en PHP para la interacción con la base de datos.

- JSON : Manipulado con JavaScript para obterner las Provincias y Ciudades de España en el formulario de registro y en los pedidos.

- MySQL : Base de datos con las tablas usuarios, articulos, carritos, pedidos y comentarios.
